什么是注释？

注释提供了一种将事件数据集成到图形中的方法。 它们在所有图形面板上可视化为垂直线条和图标 。 当您将鼠标悬停在注释图标上时，您可以获得事件文本和 活动标签。 您可以 按住CTRL或CMD在grafana上单击图形（或拖动区域）直接添加注释事件。 这些注释将存储在Grafana的注释数据库中。

查看注释文档 了解更多信息。
