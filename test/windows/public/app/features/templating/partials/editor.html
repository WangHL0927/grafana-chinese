变量做什么？

变量支持更多交互式和动态的仪表板。 而不是对服务器或传感器名称等进行硬编码 在您的度量标准查询中，您可以在其位置使用变量。 变量显示为顶部的下拉选择框 仪表板。 这些下拉列表可以轻松更改仪表板中显示的数据。 查看模板文档 模板文档 以获取更多信息。
